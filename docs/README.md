# wasm(1) Documentation

Welcome to the documentation for `wasm(1)` - unified developer tools for WebAssembly.

This directory contains comprehensive guides and documentation to help you get started and make the most of the `wasm` tool.

## Documentation Contents

- **[Authentication](authentication.md)** - Learn about authentication methods and flows for accessing container registries
- **[Configuration](configuration.md)** - Understand storage configuration and layout
- **[Usage](usage.md)** - Basic usage instructions and command reference

## Quick Start

To get started quickly:

1. Install `wasm` with `cargo install wasm` (or `cargo add wasm` for library use)
2. Review [Authentication](authentication.md) to set up registry access
3. Check [Usage](usage.md) for common commands and workflows

## Getting Help

- Run `wasm --help` for command-line help
- Visit the [API Docs](https://docs.rs/wasm) for library documentation
- Check the [GitHub repository](https://github.com/yoshuawuyts/wasm) for issues and discussions
