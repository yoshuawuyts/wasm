# Website

This website is built using [Docusaurus](https://docusaurus.io/) and published
to GitHub Pages automatically when changes are pushed to the `main` branch.

## Local Development

```bash
cd website
npm install
npm start
```

This command starts a local development server. Most changes are reflected live
without having to restart the server.

## Build

```bash
npm run build
```

This command generates static content into the `build` directory.
